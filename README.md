### Inhydro Server

**Inhydro** is an **IoT platform for smart hydroponic systems**.  
This project is a **further development of Hidropol**, redesigned with a more modular, scalable, and production-ready architecture.

Inhydro acts as the **central communication hub** between hydroponic IoT devices and edge applications.

---

## Key Features

- MQTT-based IoT communication
- Telemetry, command, and ACK handling
- Built with NestJS (Node.js)
- Modular and scalable architecture
- Defensive handling of invalid device payloads
- Ready for database and dashboard integration
